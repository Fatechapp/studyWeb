"Istilah HTML 5 digunakan sebagai jargon yang mengacu pada teknologi web yang modern, yang dikembangkan oleh WHATWG"

-https://html.spec.whatwg.org


WHATWG

  • The Web Hypertext Application Technology Working Group
  • Komunitas yang terdiri dari orang-orang yang memiliki ketertarikan pada perkembangan teknologi web
  • Siapapun bisa bergabung ke dalam WHATWG
  • Membuat standar dan pengujian untuk teknologi web
  • Dibentuk oleh orang-orang dari Apple, Mozilla Foundation, dan Opera di tahun 2004
  • Didirikan karena pada saat itu W3C tidak tertarik untuk mengembangkan HTML

Sejarah HTML5

  • Pertama kali dibuat oleh Tim Berners-Lee di CERN, pada tahun 1990
    • Hanya ada 18 tag/elemen
    • Terinspirasi dari SGML (Standard Generalized Markup Language)
    • Standar penulisan dokumen di CERN
    • Tahun 1993 diusulkan ke IETF (Internet Engineering Task Force)
  • HTML 2.0 dibuat pada tahun 1994
    • Dibuat oleh HTML Working Group, sebagai bagian dari IETF
    • Menambahkan tabel, image map, upload file
  • HTML 3.0 dibuat pada Januari 19997
    • Dikembangkan dan distandardisasi oleh W3C
    • W3C dipimpin oleh Tim Berners-Lee
    • Menghilangkan tag untuk formula matematika, blink, dan marquee
  • HTML 4.0 dibuat pada Desember 1997
    • Menawarkan 3 variasi: Strict, transitional, dan frameset
      • Strict, melarang penggunaan tag yang sudah usang
      • Transitional, memperbolehkan penggunaan tag yang sudah usang
      • Frameset, hanya memperbolehkan elemen bertipe frame
    • Banyak menghilangkan elemen style, agar diganti dengan penggunaan CSS
  • XHTML mulai dibuat pada tahun 1999
    • Versi XML dari HTML
    • Pengembangan HTML mulai ditinggalkan oleh W3C, karena fokus di XHTML
  • HTML5 mulai diusulkan pada tahun 2004
    • Mozilla & Opera mengusulkan pengembangan kembali HTML
    • Usulan tersebut ditolak karena W3C lebih memilih mengembangkan XHTML
    • Apple, Mozilla, dan Opera memutuskan untuk membuat WHATWG yang fokus ke pengembangan HTML agar kompatibel juga untuk versi sebelumnya
  • Pada tahun 2006, W3C kembali tertarik untuk mengembangkan HTML5
    • WHATWG mengijinkan agar spesifikasi HTML5-nya dipublikasikan dengan lisensi dari W3C
    • W3C kemudian bekerja sama dengan WHATWG untuk mengembangkan HTML5
  • Pada tahun 2011, kembali terjadi 'konflik'
    • W3C ingin 'menyelesaikan' HTML5
    • WHATWG ingin membuat 'living standard' untuk HTML
  • Pada tahun 2019, W3C & WHATWG menandatangani perjanjian kerja sama
    • WHATWG mengembangkan 'living standard' HTML, lewat dokumen berikut: https://html.spec.whatwg.org/
    • W3C memfasilitasi
